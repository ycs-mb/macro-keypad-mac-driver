How to test

Keep this window focused and press each key on the macro pad one at a time. If a raw keycode reaches this page, the key lights up RED — that means Karabiner is not intercepting it. If the expected action fires (app opens, shortcut works) and no raw key appears here, click the key to mark it GREEN (pass).

Tip: for shortcut keys (⌘Z, ⌘C, etc.), the remapped shortcut will fire in this page — that's a pass. The test only flags keys whose raw unremapped code arrives.

🔍 D-PAD CENTER DISCOVERY MODE: The D-pad center press code is unknown. Keep this window focused and press the center D-pad button. The event log will capture the raw code — please note down whatever appears and report it!

⚠ Keys detected raw — Karabiner is not intercepting

  1. Open Karabiner-Elements → Devices tab
  2. Find "USB Keyboard (INSTANT)" in the device list
  3. Toggle "Modify events" to ON
  4. Confirm the active profile is "INSTANT 19-Key Macro Pad" (Profiles tab)
  5. Check the rule is enabled under Complex Modifications
  6. After fixing, click Reset All and re-test
