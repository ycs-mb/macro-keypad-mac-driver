      Install on macOS
      
    

    

      

        karabiner_profile.json was saved to your Downloads. Open
        Terminal and run this one line — it moves the file into the
        project and applies it to Karabiner-Elements:
      

      
mv ~/Downloads/karabiner_profile.json ~/Developer/macro-keypad/ && bash ~/Developer/macro-keypad/install_profile.sh

      

        
        
      

      

        The installer backs up your current Karabiner config first, then activates the "INSTANT 19-Key Macro Pad"
        profile. Safe to run repeatedly.
